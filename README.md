# POS AI — Challenges

Repository to gather challenges and apply concepts studied in the **Postgraduate Program in Applied Artificial Intelligence**.

Each folder contains a standalone project exploring different AI techniques through real-world scenarios.

## Challenges

| Folder | Description | Key Concepts |
|--------|-------------|--------------|
| [M1-Class3_movies](./M1-Class3_movies) | Movie recommendations with ChromaDB | Vector search, cosine similarity, ChromaDB |

## Tech Overview

- **Vector databases**: ChromaDB
- **Embeddings**: custom numeric vectors (one-hot encoding, feature weighting)
- **Similarity search**: cosine similarity, HNSW algorithm
- **Backend**: Node.js, Express
- **Frontend**: React, Vite
